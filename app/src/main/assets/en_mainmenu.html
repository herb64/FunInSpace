The Main Menu

Search allows to search for text in either the title only or the title and the explanation text. By default, only the title is searched. You can set this in the settings dialog. The search is not case sensitive.
Use Filter to shrink the list to show only image above a rating level to see your favourites only. Filtering is also possible for video types or to show only images you have defined as wallpaper.
The Settings dialog allows to define your personal preferences. See the help page for settings for details.
Debuginfo just displays a screen with some information useful for debugging. Use this if Herbert asks you to do so. This will be removed in the final version.
The Slideshow function is not yet available. This will allow later to run a slideshow with your favourite images.
Restore Original Wallpaper allows to restore the wallpaper, which was active before installing the app. Important: if you plan to uninstall the app, run this BEFORE.
Search NASA Archive allows to load images for a a selectable time range. An optional search string can be set.
Email log to Herbert is currently just used for debugging. It would be good, if you send the data from time to time, so that I can have a look, if something went wrong. This will most likely be removed in the final version.
Display the logfile allows you to see the log contents. You also can send the log via email. And you are able to clear the log.
FileTransfer This is only for debugging and you should actually not use that function. This will definitely be removed in the final version. For those who are technically interested, please contact me.
About displays infos about the app, like version, licenses etc. in a small dialog window.
Privacy Policy This shows the privacy policy.
Help displays the help. You are encouraged to have a look at the help.

The context menu

Delete the selected item(s). Note, that the delete function does not yet have an undo.
Share the selected item(s). This is not yet available. It might be used to share items with other users via Whatsapp and Co..
Rate your personal favourite images or videos. Higher rated images are more likely to be shown with the wallpaper shuffle. You can also filter the list for images with a minimum rating to quickly access your favourites.
Start reading the explanation text.
Stop reading the explanation text.
Set as wallpaper can be used to select the current image as a wallpaper. This opens the image in fullscreen and shows the selection rectangle.
Choose new wallpaper range can be used to change the range of the wallpaper in the current image.
Remove wallpaper removes the wallpaper for the selected image(s). Note, that if the currently active wallpaper is in the selection, this will not be removed.
Remove local copy removes the local copy of the hires image stored on the device.
